
# Bank of Canada App

## How to run

### Run a maven clean 
```bash
mvn clean
```
### Run a maven install
```bash
mvn install
```
### Go into the target folder
```bash
cd target
```
### Run the command 
```bash
java -jar BankOfCanada-1.0-SNAPSHOT-shaded.jar 2019-05-09 2020-05-12 FXAUDCAD FXCADUSD
```
